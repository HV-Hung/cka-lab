# Hands-on Labs

Self-contained, reproducible labs grouped by CKA exam domain. Each lab includes manifests, validation steps, and a troubleshooting scenario.

**Quick start:** From the repo root, run `make up` then ask the agent: *"Set up the pods lab"*
